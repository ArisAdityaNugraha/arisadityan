Portfolio Details

Payroll Apps

Payroll Apps adalah sebuah Aplikasi yang dibuat dengan bahasa pemrograman PHP Framework Code Igniter dan Database Mysql

Project ini melibatkan Karyawan dan Bagian HRGA, Terdapat 2 pengguna yaitu User dan Admin, Admin mempunyai hak akses full Seperti Dashboard (Rekapitulasi seluruh data), Menu Master Data (Data Pegawai), Transaksi (Perhitungan Gaji, Potongan, Tunjangan, dll), Menu Laporan untuk melihat hasil inputan, Ubah Password untuk mengubah password. Sedangkan hak akses user hanya dapat melihat Menu laporan yang dimana dapat melihat Slip Gaji secara online dan dapat menyimpannya dalam bentuk PDF.

Project ini dibuat pada saat bekerja di PT. Asalta Mandiri Agung
